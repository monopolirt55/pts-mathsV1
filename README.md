# PTS-Maths

**Une PWA pour apprendre, s'entraîner et réviser la spécialité mathématiques en Première et Terminale.**

Inspirée de Khan Academy et Duolingo, avec la pédagogie d'Yvan Monka.

---

## ✨ Fonctionnalités

- **14 chapitres** entièrement jouables (Première spé + Bases collège/seconde)
- Plus de **300 exercices** en LaTeX (rendu mathématique professionnel)
- **Vidéos Yvan Monka** intégrées pour chaque propriété
- **Boucle d'apprentissage** complète : vidéo → explication → exercices → maîtrise → évaluation finale
- **Système de gamification** : XP, niveaux, séries quotidiennes, boutique XP
- **Feedback IA personnalisé** sur les erreurs (via clé Gemini gratuite)
- **Studio de création d'images** avec Pollinations (fonds d'écran perso)
- **Mode sombre** complet pour les sessions nocturnes
- **Animation Sans Faute** spectaculaire pour les sessions parfaites
- 100 % offline une fois installée (PWA)

## 📚 Chapitres disponibles

### Première Spécialité Maths
- Produit scalaire (25 exos)
- Géométrie repérée (25 exos)
- Suites arithmétiques et géométriques (25 exos)
- Dérivation (25 exos)
- Second degré (25 exos)
- Probabilités conditionnelles (25 exos)

### Bases (collège & seconde)
- Calcul littéral (20 exos)
- Fractions (20 exos)
- Pythagore et Thalès (20 exos)
- Puissances et racines (20 exos)
- Équations du 1er degré (20 exos)
- Trigonométrie de base (20 exos)
- Vecteurs de seconde (20 exos)
- Fonctions affines (20 exos)

## 🛠️ Stack technique

- HTML + CSS + JavaScript pur (pas de framework)
- Tailwind CSS via CDN
- KaTeX pour le rendu mathématique
- canvas-confetti pour les animations
- Pollinations.ai pour la génération d'images
- Gemini 2.0 Flash API (facultatif) pour le feedback IA

## 🚀 Lancement local

Double-clique sur `index.html` pour ouvrir dans ton navigateur.

Pour une expérience complète (vidéos YouTube, mode hors ligne, installation PWA), déploie sur GitHub Pages — voir `DEPLOY.md`.
